## 签订委托合同

### 触发方式
用户发送：【客户编号】+ "签订合同"

### 前置数据读取

从复盘台账读取：
- 客户基本信息（姓名、身份证号、电话、地址）
- 对方当事人信息
- 案由、推荐诉讼路径

### 步骤1：确认收费方式（强制询问）

**必须询问用户确认，不得跳过。**

> "请确认本案收费方式：
> A. 基础费用
> B. 基础费用 + 风险代理
> C. 风险代理"

用户确认后记录收费方式，进入下一步。

### 步骤2：确认授权范围（强制询问）

**必须询问用户确认，不得跳过。**

> "请确认授权范围类型：
> A. 一般授权
> B. 特别授权
> C. 执行授权"

**三种授权类型的权限内容**：

**A. 一般授权（5项）**：
代为调查收集证据、提供证据；2、代为出庭参加诉讼、陈述事实、进行举证和质证、参加辩论；3、代为签收法律文书；4、代为提出管辖异议、回避申请等程序性申请；5、代为申请财产保全和证据保全。

**B. 特别授权**：
代为承认、放弃、变更诉讼请求；2、代为进行和解；3、代为提起反诉或上诉；4、代为签收调解书；5、代为申请强制执行；6、代为领取标的款项及其他财物；7、代为选择鉴定、评估、拍卖等中介机构；8、代为进行诉前调解、参加庭前会议、签署送达地址确认书；9、代为办理委托人在诉讼中的全部事项，行使委托人的一切实体权利和程序权利。

**C. 执行授权（9项）**：
代为申请执行和恢复执行；2、代为进行执行和解；3、代为提出书面异议及申请复议；4、代为收取法律文书；5、代为选择拍卖评估等中介机构；6、代为追加或者变更执行当事人；7、代为执行听证；8、代为调取证据、材料；9、其他法律、法规、司法解释规定的申请执行人的实体和程序权利。

用户确认后记录授权类型，进入下一步。

### 步骤3：生成文书

收费方式和授权范围确认后，生成委托授权文件。

**生成文件清单**：
1. 授权委托书（按排版规范直接生成）
2. 法定代表人身份证明（企业委托时生成）

**文书生成方式**：
- 本 Skill 负责确认收费方式、授权范围、委托人类型和各文书正文信息。
- 先按 `references/` 目录中对应文书的排版规范生成语义 HTML。
- 最终本地 `.docx` 必须调用 `法律文书模板与导出`，走 HTML → `html_to_docx.py` → DOCX 链路；不得继续使用本 Skill 内旧 `python-docx + lxml` / Word XML 占位符替换方案。

**导出 profile 对应关系**：
| 文书 | 统一导出 profile |
|-----|----------------|
| 授权委托书 | `entrustment_authorization` |
| 法定代表人身份证明 | `legal_representative_certificate` |

**委托人类型判断**：
- 自然人：生成个人版授权委托书
- 企业/法人：生成企业版授权委托书 + 法定代表人身份证明

**保存路径**：
```
<当前事项业务文件路径>/01-委托材料/
```

### 步骤4：生成完毕提示（原样输出）

文书生成后自动输出以下完整提示：

**个人委托**：
> 授权委托书×3，另需客户提供身份证复印件×3。

**企业委托**：
> 授权委托书×3、法定代表人身份证明×3，另需客户提供法定代表人身份证复印件×3（加盖公章）、营业执照×3（加盖公章）。

**调档提示**（生成委托材料后自动提醒）：
> 📋 **【调档材料备忘】**
> 如需前往法院/仲裁委调取案卷，请提前准备：
> 1. 授权委托书（当事人已签署的原件）
> 2. 律师执业证（原件 + 复印件）
> 3. 律师事务所调查专用介绍信（向律所申请开具）
> 4. 当事人身份证明复印件（自然人：身份证×1；企业：营业执照×1 + 法定代表人身份证明×1）
> 5. 调档申请书（注明案号及调档事由）
>
> 去公安机关查询自然人户籍需要：介绍信、律师证、当事人身份证复印件、人口信息查阅申请表。

## 案件归档管理

### 触发方式
用户发送：【客户编号】+ "案件建档"

### 职责说明

- 文件夹创建由主Skill（民事一审诉讼Agent）在S0→S1阶段完成
- 本Skill负责：生成诉讼案件跟踪记录表 + 按确认更新复盘台账中的案件状态字段

### 工作流程

#### 步骤1：检查档案结构
验证客户文件夹 `<当前事项业务文件路径>/` 是否已存在。不存在则提示用户先通过主Skill建档。

#### 步骤2：生成《诉讼案件跟踪记录表》

文档结构：案件基本信息、重要时间节点、工作记录、费用收支记录、关联文档索引、案件进展备注。

保存路径：`<当前事项业务文件路径>/诉讼案件跟踪记录表_【客户姓名】.docx`

#### 步骤3：更新复盘台账

复盘台账路径：复盘台账（按法律工作总控确认后读取/更新）

回写字段：

| 字段名 | 回写内容 |
|-------|---------|
| 案件状态 | "已委托" |
| 委托日期 | 当前日期 |
| 授权类型 | "一般授权" / "特别授权" / "执行授权" |
| 案由 | 从前置文档读取 |
| 对方当事人 | 从前置文档读取 |
| 授权文件路径 | 授权委托书保存路径 |
| 案件文件夹路径 | 客户文件夹路径 |
| 收费方式 | "基础费用" / "基础费用+风险代理" / "风险代理" |

字段不存在时自动在末尾新增。系统记录同步在文档生成后自动执行。

#### 步骤4：完成提示

> "案件建档完成。跟踪记录表已生成，客户档案已更新。签署完毕后请及时更新跟踪记录表中的重要时间节点。"
